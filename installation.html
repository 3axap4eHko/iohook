# Installation

iohook provides prebuilt version for a bunch of OSes and runtime versions.

npm install iohook --save # or yarn add iohook

iohook currently provide prebuilt versions for the following runtimes:

  • Electron:

    • 1.0.X (ABI 47)
    • 1.2.X (ABI 48)
    • 1.3.X (ABI 49)
    • 1.4.X (ABI 50)
    • 1.5.X (ABI 51)
    • 1.6.X (ABI 53)
    • 1.7.X (ABI 54)
    • 1.8.X (ABI 57)
    • 2.0.X (ABI 57)

  • Node.js:

    • 4.6.X (ABI 46)
    • 5.12.X (ABI 47)
    • 6.9.X (ABI 48)
    • 7.4.X (ABI 51)
    • 8.9.X (ABI 57)
    • 9.2.X (ABI 59)
    • 10.0.X (ABI 64)
